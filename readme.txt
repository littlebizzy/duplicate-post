== Description ==

Easily duplicate (clone) any blog post, custom post, or page in WordPress which are then saved in Draft mode, saving you tons of time and headache.

Zero settings page to worry about, simply adds a "Duplicate" link to the post listings and page listings area.

== Installation ==

1. Upload to `/wp-content/plugins/` directory
2. Activate the plugin in WP Admin
